## ğŸ› ï¸ Getting Started

### 1. Clone Repository

```bash
git clone https://github.com/wku-crew/saas-crew.git
cd saas-crew
```

### 2. Install Dependencies

```bash
# Install pnpm globally
npm install -g pnpm@8

# Install project dependencies
pnpm install
```

### 3. Environment Setup

```bash
# Copy environment templates
cp apps/api/.env.example apps/api/.env
cp apps/web/.env.example apps/web/.env

# Edit .env files with your credentials
# - GitHub OAuth (create app at github.com/settings/developers)
# - Database URL (local PostgreSQL)
# - JWT secret (random string 32+ chars)
```

### 4. Database Setup

```bash
# Start PostgreSQL (if using Docker)
docker run -d \
  --name wku-crew-postgres \
  -e POSTGRES_USER=postgres \
  -e POSTGRES_PASSWORD=postgres \
  -e POSTGRES_DB=wku_crew_dev \
  -p 5432:5432 \
  postgres:16-alpine

# Generate Prisma client
pnpm --filter @wku-crew/api db:generate

# Run migrations
pnpm --filter @wku-crew/api db:push
```

### 5. Start Development Servers

```bash
# Terminal 1: API server
pnpm --filter @wku-crew/api dev

# Terminal 2: Web server
pnpm --filter @wku-crew/web dev
```

Visit:
- Frontend: http://localhost:3000
- API: http://localhost:4000
- Swagger UI: http://localhost:4000/api/docs
