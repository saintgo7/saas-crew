# WKU 소프트웨어 Crew Project - 파트너십 전략

## 1. 파트너십 개요

### 1.1 파트너십 목표
- 대학 파트너십: 10개 대학 (3년 내)
- 기업 파트너십: 20개 기업 (3년 내)
- 기술 파트너십: 5개 (1년 내)

---

## 2. 대학 파트너십

### 2.1 타겟 대학

#### 1차 (1년차): 원광대학교
- 현재 위치
- 소프트웨어학과 공식 플랫폼 채택
- 학점 연동 논의

#### 2차 (2년차): 전라북도 대학
- 전북대학교 (종합)
- 군산대학교 (공과대)
- 전주대학교 (IT 계열)

#### 3차 (3년차): 수도권 확장
- 서강대, 성균관대, 고려대 등
- 목표: 10개 대학

### 2.2 파트너십 모델

**모델 A: 공식 채택**
- 학과 공식 학습 플랫폼
- 교수 참여 (강의 제작)
- 학점 인정 가능

**모델 B: B2B 라이선스**
- 대학 전체 라이선스 구매
- 연 ₩30M
- 화이트라벨 제공

**모델 C: 협업**
- 커리큘럼 공동 개발
- 해커톤 공동 주최
- 무료 제공 (홍보 목적)

### 2.3 제안 프로세스

1. **초기 접촉**: 학과장, 교수 미팅
2. **파일럿**: 소규모 테스트 (무료)
3. **평가**: 3개월 사용 후 효과 측정
4. **계약**: 공식 파트너십 체결

---

## 3. 기업 파트너십

### 3.1 타겟 기업

#### IT 대기업
- 삼성전자, 네이버, 카카오
- 목적: 채용 연계, 인턴십

#### 스타트업
- 시리즈 B 이상
- 목적: 채용, 프로젝트 협업

#### 지역 기업
- 전북 소재 IT 기업
- 목적: 산학 협력

### 3.2 파트너십 유형

**Type 1: 채용 파트너십**
- 플랫폼 내 채용 공고 게시
- 포트폴리오 기반 매칭
- 수수료: 채용 성공 시 연봉의 10%

**Type 2: 프로젝트 협업**
- 기업 프로젝트를 학생 팀에 제공
- 실무 경험 + 과제 해결
- 수수료: 프로젝트당 ₩5-10M

**Type 3: 교육 협업**
- 기업 멘토 제공
- 현직자 특강
- 무료 (홍보 효과)

**Type 4: 스폰서십**
- 해커톤, 세미나 스폰서
- 금액: ₩5-10M/회

## 4. 기술 파트너십

### 4.1 클라우드 제공자

**AWS (Amazon Web Services)**
- **파트너십**: AWS Activate (스타트업 크레딧)
- **혜택**: $10,000 크레딧 (1년)
- **목적**: 인프라 비용 절감

**GCP (Google Cloud Platform)**
- **파트너십**: GCP for Startups
- **혜택**: $3,000 크레딧

### 4.2 개발 도구

**GitHub**
- **파트너십**: GitHub Education
- **혜택**: Pro 기능 무료
- **목적**: 학생 계정 혜택

**JetBrains**
- **파트너십**: 교육 라이선스
- **혜택**: IDE 무료 사용

### 4.3 서비스 제공자

**SendGrid (이메일)**
- 월 10만 건 무료

**Cloudflare (CDN)**
- 스타트업 플랜
