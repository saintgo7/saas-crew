# WKU 소프트웨어 Crew Project - 경쟁사 분석

## 1. 경쟁 구도 개요

### 1.1 경쟁 매트릭스

```
고가 (Premium)
    │
    │   [엘리스]          [Coursera]
    │                     [부트캠프]
    │
    │              [WKU CREW]
    │   [프로그래머스]
    │
    │   [인프런]          [Udemy]
    │                     [YouTube]
저가 (Free/Low-cost)
    │
    └───────────────────────────────
  개인학습              협업/프로젝트
```

### 1.2 경쟁 분류

#### Tier 1: 직접 경쟁자
- **엘리스 (Elice)**: 대학 LMS + 클라우드 IDE
- **프로그래머스 (Programmers)**: 코딩 테스트 + 학습

#### Tier 2: 부분 경쟁자
- **인프런 (Inflearn)**: 온라인 강의 플랫폼
- **Udemy**: 글로벌 강의 플랫폼
- **Coursera**: MOOC 플랫폼

#### Tier 3: 대체재
- **코딩 부트캠프**: 오프라인 집중 교육
- **YouTube**: 무료 교육 콘텐츠
- **대학 정규 교육**: 학위 과정

## 2. 직접 경쟁사 상세 분석

### 2.1 엘리스 (Elice)

#### 회사 개요
- **설립**: 2015년
- **본사**: 서울, 대한민국
- **직원 수**: 약 150명
- **누적 투자**: 약 300억원
- **사용자 수**: 약 50만명
- **파트너 대학**: 50개 이상

#### 비즈니스 모델
- **B2B 중심**: 대학, 기업 라이선스 판매
- **가격**: 대학 라이선스 연 5,000만원~
- **수익원**: B2B 라이선스 (90%), B2C (10%)

#### 핵심 기능
1. **클라우드 IDE "Elice Lab"**
   - 브라우저 기반 개발 환경
   - Python, C, Java 등 지원
   - 실시간 코드 실행

2. **LMS (학습 관리 시스템)**
   - 과제 제출 및 자동 채점
   - 진도 관리
   - 성적 통계

3. **대학 커스터마이징**
   - 학교별 브랜딩
   - 커리큘럼 커스터마이징
   - 학사 시스템 연동

4. **AI 튜터**
   - 질문 자동 응답
   - 학습 추천

#### 강점 (Strengths)
- ✅ 대학 시장 선점 (50개 대학)
- ✅ 안정적인 클라우드 IDE
- ✅ 대학 맞춤형 기능
- ✅ 풍부한 투자 자본
- ✅ 기업 교육 경험

#### 약점 (Weaknesses)
- ❌ 높은 가격 (대학 예산 부담)
- ❌ 콘텐츠 부족 (자체 제작 콘텐츠 적음)
- ❌ 커뮤니티 부재
- ❌ 창업 지원 없음
- ❌ 학생 직접 구매 어려움 (B2B 위주)

#### 기회 (Opportunities)
- 기업 교육 시장 확대
- AI 기술 고도화
- 해외 시장 진출

#### 위협 (Threats)
- 글로벌 플랫폼 경쟁
- 클라우드 비용 증가
- 무료 대체재 증가

#### 시장 포지셔닝
- **타겟**: B2B (대학, 기업)
- **가격**: 프리미엄
- **차별화**: 클라우드 IDE + LMS 통합

## 4. 대체재 분석

### 4.1 코딩 부트캠프

#### 주요 업체
**코드스테이츠**
- 기간: 6개월 (풀타임)
- 비용: ₩8,000,000-10,000,000
- 취업 연계: 취업 보장 (6개월 내)
- 환불 정책: 취업 실패 시 전액 환불

**패스트캠퍼스**
- 기간: 3-6개월
- 비용: ₩3,000,000-7,000,000
- 온라인 + 오프라인 혼합
- 강사: 현직 개발자

#### 강점 vs 약점
**강점**:
- 집중 교육 (몰입)
- 높은 취업률
- 현직자 멘토링
- 강제성 (동기부여)

**약점**:
- 매우 비싼 비용
- 시간 투자 큼 (풀타임)
- 지역 제한 (서울 중심)
- 높은 강도 (중도 포기 위험)

#### 우리의 포지셔닝
- **보완재**: 부트캠프 전 기초 학습
- **대체재**: 부트캠프 대신 저렴하게
- **차별화**: 학업과 병행 가능, 저비용

- 학점 연동 가능성

- 학내 네트워킹

**경쟁사와 차이**:
- 엘리스: B2B 위주, 학생 직접 접근 어려움
- 인프런: 일반 플랫폼, 대학 연계 없음

#### 2. 올인원 통합 플랫폼
**제공 기능**:
- 학습 (LMS)
- 개발 (Cloud IDE)
- 협업 (Real-time collaboration)
- 배포 (CI/CD)
- 커뮤니티 (Q&A, 멘토링)

**경쟁사와 차이**:
- 엘리스: 개발 환경만
- 인프런: 학습만
- 프로그래머스: 알고리즘만

#### 3. 단계별 성장 시스템
**Junior → Senior → Master**
- 명확한 레벨 구분
- 레벨업 조건 명시
- 보상 시스템 (배지, 인증서)
- 레벨별 멘토 매칭

**경쟁사와 차이**:
- 대부분 플랫폼: 레벨 구분 없음
- Coursera: 단일 코스 완료 개념

#### 4. 창업 생태계
**제공 서비스**:
- 아이디어 검증
- 팀 빌딩
- 프로토타입 개발
- 투자 연계
- 법률/회계 지원

**경쟁사와 차이**:
- 모든 경쟁사: 창업 지원 없음
- 우리만의 독보적 영역

### 5.3 진입 장벽 구축

#### 네트워크 효과

- **원광대 공식**: 신뢰도 및 권위
- **성공 사례**: 졸업생 창업 스토리
- **커뮤니티 문화**: 독특한 학습 문화

## 6. 경쟁 전략

### 6.1 포터의 5 Forces 분석

#### 1. 산업 내 경쟁 (Rivalry)
**강도**: ⭐⭐⭐☆☆ (중간)
- 다수의 플레이어 존재
- 하지만 각자 다른 니치
- 우리는 대학+창업 니치

**전략**: 차별화 및 니치 집중

#### 2. 신규 진입 위협 (New Entrants)
**강도**: ⭐⭐⭐⭐☆ (높음)
- 기술 진입 장벽 낮음
- 클라우드 인프라 접근 용이
- 콘텐츠 제작 비교적 쉬움

**전략**: 빠른 네트워크 효과 구축

#### 3. 대체재 위협 (Substitutes)
**강도**: ⭐⭐⭐⭐⭐ (매우 높음)
- YouTube 무료 콘텐츠
- 부트캠프
- 정규 대학 교육

**전략**: 올인원 가치 제안으로 차별화

#### 4. 구매자 교섭력 (Buyer Power)
**강도**: ⭐⭐⭐⭐☆ (높음)
- 학생 예산 제한적
- 다수의 대체재 존재
- 전환 비용 낮음

**전략**: 가격 경쟁력 + 락인 효과

#### 5. 공급자 교섭력 (Supplier Power)
**강도**: ⭐⭐☆☆☆ (낮음)
- 교수 및 강사 다수
- 클라우드 제공자 다수
- 오픈소스 도구 활용

**전략**: 다양한 공급원 확보

### 6.2 경쟁 전략 프레임워크

#### Blue Ocean Strategy
**제거 (Eliminate)**
- 불필요한 마케팅 비용
- 과도한 강의 콘텐츠
- 복잡한 인터페이스

**감소 (Reduce)**
- 강의 영상 길이
- 가격 (학생 부담)
- 기능 복잡도

**증가 (Raise)**
- 실습 비중
- 협업 기능
- 커뮤니티 활동
- 멘토링 품질

**창조 (Create)**
- 대학 통합 생태계
- 레벨 시스템
- 창업 지원 프로그램
- 실시간 협업 환경

#### 차별화 전략 (Differentiation)
1. **대학 특화**: 학점 연동, 교수 참여
2. **협업 중심**: 혼자가 아닌 팀 학습
3. **창업 생태계**: 학습부터 창업까지

#### 원가 우위 전략 (Cost Leadership)
1. **오픈소스 활용**: 개발 비용 절감
2. **클라우드 최적화**: 인프라 비용 최소화
3. **커뮤니티 콘텐츠**: 사용자 생성 콘텐츠

### 6.3 경쟁사별 대응 전략

#### vs 엘리스
**전략**: 가격 및 접근성 우위
- 학생 직접 구매 가능한 저가 모델
- 커뮤니티 중심 학습
- 창업 지원으로 차별화

**포지셔닝**: "학생을 위한 엘리스"

#### vs 프로그래머스
**전략**: 초보자 친화성
- 입문자부터 시작 가능
- 프로젝트 중심 학습
- 협업 경험 제공

**포지셔닝**: "취업 전 실력을 키우는 곳"

#### vs 인프런
**전략**: 통합 환경
- 학습+실습+배포 올인원
- 체계적 커리큘럼
- 커뮤니티 강화

**포지셔닝**: "보기만 하지 말고 직접 만들어보세요"

#### vs 부트캠프
**전략**: 유연성 및 비용
- 학업과 병행 가능
- 1/10 비용
- 자기 속도 학습

**포지셔닝**: "부담 없이 시작하는 코딩"
