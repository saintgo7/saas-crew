# WKU 소프트웨어 Crew Project - 부록 및 참고자료

## 1. 용어 정의 (Glossary)

### 비즈니스 용어

| 용어 | 정의 |
|------|------|
| **MVP** | Minimum Viable Product, 최소 기능 제품 |
| **PMF** | Product-Market Fit, 제품-시장 적합성 |
| **SaaS** | Software as a Service, 서비스형 소프트웨어 |
| **B2B** | Business to Business, 기업 간 거래 |
| **B2C** | Business to Consumer, 기업-소비자 거래 |
| **MRR** | Monthly Recurring Revenue, 월간 반복 수익 |
| **ARR** | Annual Recurring Revenue, 연간 반복 수익 |
| **CAC** | Customer Acquisition Cost, 고객 획득 비용 |
| **LTV** | Lifetime Value, 고객 생애 가치 |
| **Churn** | 이탈률 |
| **DAU** | Daily Active Users, 일간 활성 사용자 |
| **MAU** | Monthly Active Users, 월간 활성 사용자 |
| **NPS** | Net Promoter Score, 순 추천 고객 지수 |
| **KPI** | Key Performance Indicator, 핵심 성과 지표 |
| **OKR** | Objectives and Key Results, 목표 및 핵심 결과 |

### 기술 용어

| 용어 | 정의 |
|------|------|
| **IDE** | Integrated Development Environment, 통합 개발 환경 |
| **CI/CD** | Continuous Integration/Continuous Deployment, 지속적 통합/배포 |
| **API** | Application Programming Interface |
| **REST** | Representational State Transfer |
| **JWT** | JSON Web Token |
| **OAuth** | Open Authorization |
| **CDN** | Content Delivery Network |
| **VPC** | Virtual Private Cloud |
| **CRDT** | Conflict-free Replicated Data Type |
| **SSR** | Server-Side Rendering |
| **CSR** | Client-Side Rendering |

## 10. FAQ (자주 묻는 질문)

### Q: 이 프로젝트는 언제 시작하나요?
A: 2026년 1월 22일 공식 킥오프, 4월 정식 런칭 예정입니다.

### Q: 투자 유치는 어떻게 진행되나요?
A: Pre-Seed (현재), Seed (1년차), Series A (2년차) 순으로 진행됩니다.

### Q: 수익 모델은 무엇인가요?
A: Freemium (무료 + 유료 구독) + B2B 라이선스 + 정부 지원금입니다.

### Q: 경쟁사 대비 강점은?
A: 대학 특화, 올인원 플랫폼, 창업 지원이 차별화 포인트입니다.

### Q: 팀은 어떻게 구성되나요?
A: CEO, CTO, CPO, CMO, COO 아래 개발, 디자인, 마케팅, 운영 팀으로 구성됩니다.
