# WKU Software Crew Project - Project Overview

## 1. Executive Summary

### 1.1 Project Name
**WKU Software Crew** - All-in-One Learning & Startup Platform for University Students

### 1.2 Vision
"Democratizing software education and empowering student entrepreneurs through technology"

### 1.3 Mission
- Provide accessible, high-quality programming education for all students
- Bridge the gap between academic learning and industry requirements
- Support students from learning to career/entrepreneurship
- Build a thriving community of student developers and entrepreneurs

### 1.4 One-liner
"The complete platform where university students learn to code, build projects, and launch startups"

---

## 2. Project Background

### 2.1 Problem Statement

**Current Pain Points**:

1. **Learning Challenges**:
   - Expensive programming courses (₩300,000-₩1,000,000)
   - Theory-heavy curriculum disconnected from practical skills
   - Lack of hands-on project experience
   - Limited access to quality learning resources

2. **Development Environment**:
   - Complex local setup (IDE, dependencies, environment configuration)
   - Collaboration difficulties in team projects
   - No integrated deployment pipeline
   - Version control learning curve

3. **Career/Entrepreneurship Gap**:
   - Insufficient practical experience for job market
   - Lack of mentorship and guidance
   - Limited networking opportunities
   - Barriers to entrepreneurship (funding, knowledge, support)

### 2.2 Solution

**WKU Software Crew Platform** - An integrated solution providing:

- **Cloud-based Learning Management System (LMS)**
- **Browser-based IDE with real-time collaboration**
- **One-click project deployment**
- **Community Q&A and mentoring system**
- **Startup incubation support**

## 5. Product Structure

### 5.1 Three-Tier Class System

#### **Junior Class** (Beginner Level)
**Target**: Freshmen and programming beginners

**Curriculum**:
- Programming fundamentals (Python, JavaScript)
- Web development basics (HTML, CSS)
- Introduction to algorithms and data structures
- First project experience

**Duration**: 3-6 months

#### **Senior Class** (Intermediate Level)
**Target**: Students with basic programming knowledge

**Curriculum**:
- Advanced frameworks (React, Node.js)
- Full-stack development
- Database design and management
- Team project collaboration
- DevOps basics (CI/CD, Docker)

**Duration**: 6-12 months

#### **Master Class** (Advanced + Entrepreneurship)
**Target**: Advanced students and aspiring entrepreneurs

**Curriculum**:
- System architecture and design
- Advanced algorithms and optimization
- Cloud infrastructure (AWS, Azure)
- Startup fundamentals
- Business model development
- Pitch deck creation
- Funding and investor relations

**Duration**: 12-24 months

### 5.2 Platform Features

#### **Learning Management System (LMS)**:
- Video lectures with interactive quizzes
- Progress tracking and certificates
- Gamification (badges, levels, leaderboards)
- Personalized learning paths

#### **Cloud IDE**:
- Monaco Editor (VS Code engine)
- 20+ language support
- Real-time collaboration (Y.js)
- Integrated terminal
- File management system

#### **Deployment System**:
- One-click deployment to cloud
- Support for static sites, Node.js, Python apps
- Custom domain support
- HTTPS certificates
- Preview environments

#### **Community & Mentoring**:
- Q&A forum (Stack Overflow style)
- Study group matching
- 1:1 mentoring sessions
- Peer code reviews
- Project showcases

#### **Startup Support**:
- Incubation program
- Mentorship from industry professionals
- Funding application assistance
- Legal and accounting consultations
- Demo day opportunities
