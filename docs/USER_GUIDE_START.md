## 회원가입 방법

WKU Software Crew는 GitHub 계정을 통해 간편하게 가입할 수 있습니다.

### Step 1: GitHub OAuth 인증

1. WKU Software Crew 홈페이지에 접속합니다.
2. 우측 상단의 **"로그인"** 또는 **"시작하기"** 버튼을 클릭합니다.
3. **"GitHub로 로그인"** 버튼을 선택합니다.

```
[스크린샷 자리: 홈페이지 로그인 버튼]
```

### Step 2: GitHub 계정 연동

1. GitHub 로그인 페이지로 이동됩니다.
2. GitHub 계정 정보를 입력합니다.
   - 이미 GitHub에 로그인되어 있다면 자동으로 다음 단계로 넘어갑니다.
3. WKU Software Crew가 요청하는 권한을 확인합니다.
   - 이메일 주소 조회
   - 프로필 정보 조회
   - 공개 저장소 정보 조회

```
[스크린샷 자리: GitHub OAuth 권한 승인 화면]
```

### Step 3: 권한 승인

**"Authorize"** 버튼을 클릭하여 권한을 승인합니다.

권한 승인 후 자동으로 WKU Software Crew 플랫폼으로 돌아옵니다.

### 가입이 안 되는 경우

#### 문제: GitHub 계정이 없는 경우
**해결방법**: GitHub 계정을 먼저 생성해야 합니다.
1. https://github.com 접속
2. 우측 상단 "Sign up" 클릭
3. 이메일, 비밀번호, 사용자 이름 입력
4. 이메일 인증 완료

#### 문제: OAuth 권한 승인 실패
**해결방법**:
1. GitHub 설정에서 WKU Software Crew 권한 확인
2. Settings > Applications > Authorized OAuth Apps
3. WKU Software Crew 권한 재승인

## 첫 로그인 절차

회원가입이 완료되면 자동으로 로그인되어 프로필 설정 페이지로 이동합니다.

### Step 1: 환영 메시지 확인

```
🎉 환영합니다!

WKU Software Crew에 가입하신 것을 축하합니다.
프로필을 완성하여 크루원으로서의 여정을 시작하세요.

[프로필 완성하기] 버튼
```

### Step 2: 기본 정보 확인

GitHub에서 가져온 정보가 자동으로 입력되어 있습니다.

- **이름**: GitHub 프로필 이름
- **이메일**: GitHub 계정 이메일
- **프로필 사진**: GitHub 아바타

이 정보는 수정할 수 있습니다.

### Step 3: 세션 유지

로그인 상태는 JWT 토큰을 통해 유지됩니다.

- **유효 기간**: 7일
- **자동 갱신**: 활동 시 자동으로 연장
- **보안 로그아웃**: 7일 이상 미접속 시 자동 로그아웃

## 프로필 설정

프로필은 크루원들에게 여러분을 소개하는 중요한 공간입니다.

### 필수 정보 입력

#### 1. 기본 정보

**이름** (필수)
- GitHub에서 가져온 이름을 사용하거나 수정
- 크루원들에게 표시될 이름
- 예시: "홍길동", "Gildong Hong"

**학과** (선택)
- 소속 학과 입력
- 예시: "컴퓨터소프트웨어공학과", "정보통신공학과"

**학년** (선택)
- 현재 학년 선택
- 옵션: 1학년, 2학년, 3학년, 4학년, 졸업생

```
[스크린샷 자리: 프로필 편집 - 기본 정보]
```

#### 2. 자기소개

**소개글** (선택)
- 자신을 표현하는 간단한 소개글
- 최대 500자
- 예시:
  ```
  백엔드 개발에 관심이 많은 2학년 학생입니다.
  Node.js와 Python을 주로 사용하며,
  REST API 개발과 데이터베이스 설계에 흥미가 있습니다.
  ```

#### 3. 프로필 사진

**아바타 이미지**
- GitHub 프로필 사진이 기본으로 설정됩니다.
- 변경하려면 **"사진 변경"** 버튼 클릭
- 지원 형식: JPG, PNG, GIF
- 최대 크기: 5MB
- 권장 크기: 500x500px

### 선택 정보 입력

#### 테마 설정

**색상 테마**
- Light (라이트): 밝은 배경
- Dark (다크): 어두운 배경
- System (시스템): 운영체제 설정 따름

**언어 설정**
- 한국어 (기본값)
- English (영어)

```
[스크린샷 자리: 프로필 편집 - 환경 설정]
```

#### 소셜 링크 (향후 추가 예정)

- GitHub 저장소
- 개인 블로그
- LinkedIn
- 포트폴리오 사이트

### 프로필 저장

1. 모든 정보를 입력한 후 **"저장"** 버튼 클릭
2. 성공 메시지 확인: "프로필이 업데이트되었습니다."
3. 프로필 페이지로 이동하여 확인

## 레벨 시스템 이해하기

WKU Software Crew는 게임처럼 레벨을 올리며 성장하는 시스템을 사용합니다.

### 3단계 랭크 시스템

```
┌─────────────────────────────────────────────────┐
│                                                 │
│  Junior (주니어)     →  Senior (시니어)          │
│  Level 1-10             Level 11-30             │
│  기초 학습              중급 개발자              │
│                                                 │
│                     →  Master (마스터)          │
│                        Level 31-50              │
│                        고급 개발자              │
│                                                 │
└─────────────────────────────────────────────────┘
```

### Junior (주니어) - Level 1-10

**목표**: 프로그래밍 기초 다지기

**특징**:
- 기본 문법과 개념 학습
- 튜토리얼 코스 수강
- 간단한 프로젝트 제작
- 커뮤니티에서 질문하기

**권장 활동**:
- 기초 코스 완강 (HTML, CSS, JavaScript)
- 첫 번째 개인 프로젝트 완성
- 10개 이상의 질문 또는 답변 작성
- GitHub 저장소 만들기

**레벨업 조건**:
- 코스 완료로 XP 획득
- 프로젝트 등록으로 XP 획득
- 커뮤니티 활동으로 XP 획득

### Senior (시니어) - Level 11-30

**목표**: 실전 프로젝트 경험 쌓기

**특징**:
- 풀스택 개발 능력
- 팀 프로젝트 참여
- 멘토링 활동 시작
- 커뮤니티 기여

**권장 활동**:
- 중급 코스 수강 (React, Node.js, Database)
- 팀 프로젝트 2개 이상 완성
- 주니어 멘토링
- 베스트 답변 5개 이상

**새로운 권한**:
- 멘토링 자격 획득
- 크루 이벤트 우선 참여
- 프로젝트 쇼케이스 featured 등록 가능

### Master (마스터) - Level 31-50

**목표**: 크루 리더 및 기업 연계

**특징**:
- 고급 기술 스택 마스터
- 크루 리더 역할
- 기업 프로젝트 참여 기회
- 플랫폼 운영 참여

**권장 활동**:
- 고급 코스 수강 (Architecture, DevOps, Cloud)
- 기업 연계 프로젝트 참여
- 세미나 발표 및 워크샵 진행
- 크루 운영진 활동

**특별 혜택**:
- 기업 인턴십 추천
- 프로젝트 매칭 우선권
- 전문가 멘토 연결
- 크루 행사 기획 권한

### XP (경험치) 획득 방법

XP는 다양한 활동을 통해 획득할 수 있습니다.

| 활동 | 획득 XP | 비고 |
|------|---------|------|
| **코스 학습** | | |
| 챕터 완료 | 10-50 XP | 난이도에 따라 차등 |
| 코스 완강 | 100-500 XP | 코스 레벨에 따라 차등 |
| **프로젝트** | | |
| 프로젝트 등록 | 100 XP | 최초 등록 시 |
| 프로젝트 업데이트 | 20 XP | 주 1회 제한 |
| 프로젝트 좋아요 받기 | 5 XP | 받을 때마다 |
| **커뮤니티** | | |
| 질문 작성 | 10 XP | |
| 답변 작성 | 20 XP | |
| 베스트 답변 선정 | 50 XP | |
| 투표 받기 (Upvote) | 2 XP | |
| 댓글 작성 | 5 XP | |

### 레벨업 필요 XP

각 레벨마다 필요한 XP가 증가합니다.

```
Level 1 → 2:   100 XP
Level 2 → 3:   200 XP
Level 3 → 4:   300 XP
...
Level 10 → 11: 1,000 XP (Senior 승급)
...
Level 30 → 31: 5,000 XP (Master 승급)
```

**참고**: 구체적인 수치는 운영 중 조정될 수 있습니다.

## 대시보드 둘러보기

첫 로그인 후 대시보드에서 플랫폼의 주요 기능을 한눈에 확인할 수 있습니다.

### 대시보드 구성 요소

#### 1. 내 프로필 위젯

좌측 상단에 위치하며 나의 현재 상태를 보여줍니다.

```
┌──────────────────────────────┐
│  [프로필 사진]                │
│  홍길동 (Junior)              │
│  Level 3                      │
│  ▓▓▓▓▓░░░░░ 450/800 XP       │
│                               │
│  프로필 보기 | 프로필 편집     │
└──────────────────────────────┘
```

**표시 정보**:
- 프로필 사진
- 이름 및 랭크
- 현재 레벨
- XP 진행률 (바 그래프)
- 다음 레벨까지 필요한 XP

#### 2. 내 프로젝트 목록

진행 중인 프로젝트를 확인할 수 있습니다.

```
┌──────────────────────────────┐
│  내 프로젝트                  │
│  ────────────────────────── │
│  📁 할 일 관리 앱             │
│     진행중 · 2일 전 업데이트  │
│                               │
│  📁 날씨 API 프로젝트         │
│     완료 · 1주 전             │
│                               │
│  [+ 새 프로젝트 만들기]       │
└──────────────────────────────┘
```

**기능**:
- 프로젝트 목록 확인
- 프로젝트 상태 (진행중, 완료, 보류)
- 최근 업데이트 시간
- 새 프로젝트 빠른 생성

#### 3. 코스 진행 상황

현재 수강 중인 코스 진도를 확인합니다.

```
┌──────────────────────────────┐
│  수강 중인 코스               │
│  ────────────────────────── │
│  React 기초                   │
│  ▓▓▓▓▓▓░░░░ 60% (6/10 챕터)  │
│                               │
│  Node.js 백엔드 개발          │
│  ▓▓░░░░░░░░ 20% (2/10 챕터)  │
│                               │
│  [코스 둘러보기]              │
└──────────────────────────────┘
```

**표시 정보**:
- 코스 제목
- 진행률 (퍼센트 및 바)
- 완료한 챕터 수 / 전체 챕터 수

#### 4. 활동 피드

크루원들의 최근 활동을 확인합니다.

```
┌──────────────────────────────┐
│  크루 활동 피드               │
│  ────────────────────────── │
│  🎉 김철수님이 Level 10에      │
│      도달했습니다! (Senior!)   │
│      1시간 전                  │
│                               │
│  📝 이영희님이 새 질문을 올렸습니다 │
│      "React Hook 사용법"       │
│      3시간 전                  │
│                               │
│  💡 박민수님이 프로젝트를 공개했습니다 │
│      "날씨 앱"                 │
│      5시간 전                  │
│                               │
│  [더 보기]                    │
└──────────────────────────────┘
```

**활동 유형**:
- 레벨업 소식
- 새로운 질문/답변
- 프로젝트 공개
- 코스 완강
- 베스트 답변 선정

#### 5. 레벨 진행률 차트

레벨업 진행 상황을 시각적으로 확인합니다.

```
[스크린샷 자리: 레벨 진행률 원형 차트 또는 막대 그래프]
```

**차트 정보**:
- 현재 XP
- 다음 레벨까지 필요한 XP
- 주간 획득 XP
- 월간 획득 XP

### 대시보드 활용 팁

#### 매일 확인하기
- 새로운 활동 피드 체크
- 수강 중인 코스 진도 확인

- 레벨업한 크루원 축하하기
- 자신의 성장 과정 기록하기

## 자주 묻는 질문

### Q1. GitHub 계정 없이 가입할 수 있나요?

**A**: 현재는 GitHub OAuth만 지원합니다. GitHub는 무료로 가입할 수 있으며, 개발자라면 필수적인 플랫폼이므로 먼저 GitHub 계정을 만드시는 것을 추천합니다.

### Q2. 프로필 정보를 비공개로 설정할 수 있나요?

**A**: 기본 정보(이름, 레벨, 랭크)는 크루원들에게 공개됩니다. 학과와 학년 정보는 선택 사항이며, 입력하지 않으면 표시되지 않습니다. 향후 세부적인 공개 설정 기능이 추가될 예정입니다.

### Q3. 레벨은 어떻게 빨리 올릴 수 있나요?

**A**: 다양한 활동을 골고루 하는 것이 좋습니다.
- 코스 학습으로 체계적인 XP 획득
- 프로젝트 등록으로 큰 XP 획득
- 커뮤니티 활동으로 꾸준한 XP 획득

### Q4. XP를 잃을 수 있나요?

**A**: 아니요. 한번 획득한 XP는 감소하지 않습니다. 다만, 부적절한 활동(스팸, 악의적 행위)이 적발되면 해당 활동으로 얻은 XP가 회수될 수 있습니다.

### Q5. 프로필 사진을 변경했는데 반영이 안 됩니다.

**A**: 브라우저 캐시 문제일 수 있습니다.
1. 페이지를 새로고침하세요 (Ctrl+F5 또는 Cmd+Shift+R)
2. 여전히 안 되면 브라우저 캐시를 삭제하세요
3. 그래도 해결되지 않으면 관리자에게 문의하세요

### Q6. 계정을 삭제하려면 어떻게 하나요?

**A**: 현재는 관리자에게 이메일로 요청해야 합니다. 향후 설정 페이지에서 직접 탈퇴할 수 있는 기능이 추가될 예정입니다.
